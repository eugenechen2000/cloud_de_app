Public test pack: pasilla RNA-seq counts

Source count matrix:
https://raw.githubusercontent.com/bioc/pasilla/master/inst/extdata/pasilla_gene_counts.tsv

Groups:
- treated (n=3)
- untreated (n=4)

Use in app:
1) Create run
2) Input type: raw_counts_matrix
3) Upload sample_sheet.csv and count_matrix.csv
4) Start run with group_a=untreated, group_b=treated (or leave blank)
